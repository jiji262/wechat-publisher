How to use

命令行选择主题:

python3 scripts/html_converter.py article.md --theme refined-blue -o article.html

列出全部主题:

python3 scripts/html_converter.py article.md --list-themes

在 accounts.yaml 中绑定账号默认主题:

theme: refined-blue · 随后 publish.py --account main 自动使用该主题。

当前默认绑定: main → refined-blue · tech → minimal-mono。其余 4 套为可选,写作时按内容类型选择。
